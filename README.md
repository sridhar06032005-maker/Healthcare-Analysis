# 🏥 Healthcare Analysis Dashboard

## 📌 Project Overview
This project presents a Healthcare Analysis Dashboard that provides insights into patient demographics, revenue, medical conditions, hospital billing, and admission types.

The dashboard helps in understanding:
- Total patients
- Revenue generation
- Average patient age
- Emergency cases
- Gender distribution
- Blood group distribution
- Medical condition trends
- Hospital billing comparison

---

## 📊 Dashboard Highlights

### 🔢 Key Metrics
- Total Patients: 56K
- Total Revenue: 1.42 Billion
- Average Age: 51.54
- Emergency Patients: 18K

### 📈 Analysis Sections
- Patient Name by Gender
- Admission Type by Patient
- Billing Amount by Hospital
- Patient Distribution by Blood Type
- Patient Distribution by Medical Condition
